# testmongoose

### software version

- g++  >= 7.4
- cmake >= 3.19.3
- git >= 2.7.4
- mongoose verion is 7.1

### Using

~~~
git clone https://github.com/SunnyWeiLucky/testmongoose.git
cd testmongoose
mkdir build
cmake ..
make -j5

./net   //start a server
./net filename  //client to upload file
~~~
